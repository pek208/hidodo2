권장 스마트 기기 사양

하이도도 학습을 원활하게 하기 위한 스마트폰 또는 태블릿PC환경입니다.

# 안드로이드

  • 하드웨어
    • [화면해상도] 390 x844 이상 권장
    • [네트워크] 100Mbp 이상 또는 개인 와이파이 환경
  • 소프트웨어
    • [운영체제] 안드로이드 10 이상
    • [기타] 안드로이드 웹뷰 지원 필수
  • 출시된 지 5년 이상 된 스마트 기기는 대부분 OS 버전이 낮거나 지원이 종료되어 하이도도 App 사용이 제한될 수 있으므로 이용 전 반드시 확인 바랍니다.
  • 특수 목적을 위해 제조된 전용 태블릿PC는 몇 가지 OS 기능을 제거하고 출시되는 경우가 대부분이기 때문에 하이도도 App을 설치 및 사용을 할 수 없습니다.

# iOS

  • 하드웨어
    • [화면해상도] 390 x844 이상 권장
    • [네트워크] 100Mbp 이상 또는 개인 와이파이 환경
  • 소프트웨어
    • [운영체제] iOS 14.0 이상
    • [기타] 학습 시 사운드를 재생하려면 매너모드 해제 확인
  • 아이폰 5와 동일한 기종이거나 이전에 출시된 모델은 지원되지 않습니다.
  • 아이패드 1, 2, 3(Air 초기 모델)은 지원되지 않습니다.
